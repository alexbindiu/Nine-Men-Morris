# Nine Men's Morris
Nine Men's Morris is a console-based game app that uses some advanced algorithms for decision-making, applied in AI.

## Description
This is a classic console game written in Python, known as "Nine Men's Morris", a.k.a. "MoarÄƒ". The game features all the options of the original game.

## Key Features
- **Play against the CPU**: Play against an advanced "AI". The decision-making is based on Minimax and Alpha-Beta Pruning algorithms (known as important algorithms used in AI developement), to provide a challenging and strategic gameplay experience.
- **Two-player mode**: Play against another person on the same device.

The game offers an authentic experience, replicating all aspects of the original Nine Men's Morris game.
